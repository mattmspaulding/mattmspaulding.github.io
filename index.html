Millie

An automation bot that notifies users when items are in stock.

Millie
Frontend EngineerCapstone Project

Summary

Worked in a team of 3 students and created a desktop application that scrapes product information from online stores and notifies users when their selected items are in stock.

Responsibilities
  • Designed and developed the application's frontend, utilizing threading to create a dynamic and responsive interface.
  • Used MongoDB to connect the frontend and backend to create a persistent and secure user-experience.
  • In an Agile development cycle, took responsibility for arranging meetings, managing requirements, and keeping our team on task for stakeholder meetings.

Pulsar Classifier

A machine-learning classifier that detects potential pulsars.

Pulsar Classifier
CreatorSolo Project

Summary

Built a classifier using machine learning models that detects potential pulsars using the HTRU2 dataset, allowing rapid and accurate analysis.

Responsibilities
  • Analyzed, cleaned, and visualized the dataset's features and correlations, then created a logistic regression and KNN model that reliably predicts potential pulsars.
  • Tested each model using a confusion matrix, f1-score, false positive rate, and a classification report.
